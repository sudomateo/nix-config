= Matthew Sanabria's Nix Configuration

Nix configuration to manage multiple machines declaratively.

== Requirements

* https://github.com/DeterminateSystems/nix-installer[Determinate Nix]

== Machines

[cols="1,1,1"]
|===
| Name | System | Type

| ms-macbook-air
| aarch64-darwin
| nix-darwin
|===

== Usage

Enter the development shell.

[source,shell]
----
nix develop
----

Apply the configuration for the current machine.

[source,shell]
----
apply
----

Or, without entering the shell first.

[source,shell]
----
nix develop --command apply
----

Format all Nix files.

[source,shell]
----
nix fmt
----

== Bootstrap

For a new macOS machine, run the following to bootstrap.

[source,shell]
----
nix run nix-darwin -- switch --flake .#ms-macbook-air
----
